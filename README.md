Video Tools
===========

Contains linux video editors tools:
* kdenlive-proxy-gopro.py - link gopro lrv files as proxies for kdenlive
* flowblade-proxy-gopro.py - link gopro lrv files as proxies for flowblade

Requirements:
-------------
* python 2.7
